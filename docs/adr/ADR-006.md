# ADR-006: Dual Licensing (Apache-2.0 OR MIT)

## Status
Accepted

## Context
covguard needs to be usable in both open-source and commercial contexts. The Rust ecosystem commonly uses dual licensing (Apache-2.0 OR MIT) to allow maximum compatibility.

## Decision
We adopt dual licensing with Apache-2.0 OR MIT for all crates.

### Rationale
- **Apache-2.0**: Well-established, patent-friendly
- **MIT**: Permissive, maximum compatibility
- **Dual**: Users can choose either license

## Consequences

### Positive
- Maximum compatibility with Rust ecosystem
- No license conflicts with downstream users
- Users can choose their preferred license

### Negative
- Slightly more complex licensing
- Need to include both LICENSE files
