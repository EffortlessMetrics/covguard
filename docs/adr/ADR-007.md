# ADR-007: Built-in Configuration Profiles

## Status
Accepted

## Context
covguard needs sensible defaults for different use cases:
- Open-source projects: Lenient, community-focused
- Team projects: Balanced, team-focused
- Strict environments: High coverage requirements

## Decision
We provide 5 built-in profiles selectable via `--profile` flag:

### Profiles
1. **oss**: 80% threshold, warn-or-error, community-focused
2. **team**: 70% threshold, warn-or-error
3. **strict**: 95% threshold, error
4. **lenient**: 50% threshold, warn-or-error
5. **moderate**: 75% threshold, warn-or-error

## Consequences

### Positive
- Quick setup for common use cases
- Sensible defaults
- Easy to customize via config file

### Negative
- Maintenance of multiple profiles
- Potential confusion about which to use
