# ADR-008: Ignore Directives

## Status
Accepted

## Context
Some code may be temporarily uncovered for valid reasons:
- Dead code being removed in a follow-up PR
- External dependencies
- Generated code

Users need a way to opt out specific lines from coverage checks.

## Decision
We support `// covguard: ignore` comment directives.

### Syntax
- Line-level: `// covguard: ignore` on the line to ignore
- Block-level: `// covguard: ignore-start` and `// covguard: ignore-end`

## Consequences

### Positive
- Fine-grained control over coverage checks
- Temporary exclusions for valid reasons
- Clear documentation of ignored lines

### Negative
- Potential for abuse (hiding real coverage issues)
- Additional parsing complexity
- Comments must be preserved in code
